# Changelog

All notable changes to this project are documented here.

## 0.1.0 - 2026-07-20

- Added deterministic parsing of public fixture data and official CBF schedule PDFs.
- Added protected real-data refresh with atomic writes and source manifests.
- Added daily previews, configurable favorite-team highlighting, and pregame alerts.
- Added local alert plans, ledger-based idempotency, and offline diagnostics.
- Added installable CLI, package resources, public documentation, and CI.
